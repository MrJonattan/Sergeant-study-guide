## P.G. 212-99 — AMBER Alert System

### Activation Criteria
- Child must be **17 or younger**.
- Police must believe the child is in **imminent danger** of serious bodily harm or death.
- May be activated for abduction by vehicle, foot, or any other means.

### Who May Request Activation
- Only a **Captain or above** from the **Detective Bureau** may request activation through the NYS Police Communications Section.

> **Exam Alert:** Only a **Captain or above from Detective Bureau** may request AMBER Alert activation. Not a patrol captain — specifically Detective Bureau.

### Activation Process

| Step | Responsibility |
|------|---------------|
| Notify NYS Police | Detective Bureau Duty Captain |
| Notify Operations Unit | Detective Bureau Duty Captain |
| Broadcast description over radio divisions | Communications Section |
| NCIC File 6 completed | Investigative unit |
| NYSPIN File 11A (Abduction Message) | Generated automatically |
| Major Case Unit notified | Detective Bureau |

### Vehicle Involvement
- If a vehicle is involved: special radio message regarding **License Plate Reader System** — manually enter the plate number.
- If vehicle description AND plate available: fax to **Traffic Management Center** for Variable Message Signs.

### Cancellation

| Timeframe | Method |
|-----------|--------|
| Within **8 hours** | Fax to NYS Police |
| After 8 hours | Via **DCPI** media notification |

> **Exam Alert:** AMBER Alert cancellation within **8 hours** is by fax to NYS Police. After 8 hours, cancellation is through DCPI media notification. The Detective Bureau Duty Captain ensures proper cancellation.

### Non-Eligible Cases
- **DCPI** available 24/7 for cases not eligible for AMBER Alert activation.

## P.G. 212-108 — NCIC/VGTOF Notifications

### VGTOF — Violent Gang and Terrorist Organization File

### Handling Codes

| Code | Meaning | Action |
|------|---------|--------|
| **Handling Code 1** | **ARREST** | Warrant exists — approach with caution |
| **Handling Code 2** | **DETAIN** | Reasonable suspicion — active investigation |
| **Handling Code 3** | **INVESTIGATION** | Do NOT alert individual — possible terrorism ties |
| **Handling Code 4** | **INFORMATION GATHERING** | Do NOT alert individual |

> **Exam Alert:** Memorize all four VGTOF Handling Codes. Code 1 = Arrest (warrant), Code 2 = Detain (reasonable suspicion), Code 3 = Investigation (do not alert, terrorism), Code 4 = Information gathering (do not alert).

### No Hit but Terrorism Suspected
- Contact **Intelligence Division Criminal Intelligence Section**.

### FBI Unable to Confirm
- Do **NOT** arrest unless probable cause exists for a federal, state, or local law violation.

## P.G. 212-129 — Facial Recognition Technology

### Fundamental Principle

> **Exam Alert:** Facial recognition does **NOT** establish probable cause by itself. It generates **investigative leads only**.

### Authorized Uses (ONLY)
1. Basis to believe person committed, is committing, or is about to commit a crime
2. Missing person, victim, or witness
3. Deceased person identification
4. Person incapacitated or unable to self-identify
5. Arrestee without valid ID
6. Imminent threat to health or public safety

### Key Definitions

| Term | Definition |
|------|-----------|
| **Probe Image** | Image of an unidentified person from witnesses, victims, or reliable sources |
| **Photo Repository** | **ONLY** arrest and parole photographs — the approved database |
| **Possible Match Candidate** | An investigative lead ONLY — does **NOT** constitute probable cause |

> **Exam Alert:** The photo repository contains **only** arrest and parole photographs. No other image sources are in the approved database.

### Process Flow
1. Investigator submits probe image to **RTCC-FIS** (Facial Identification Section).
2. FIS assigns a tracking number.
3. Investigator confirms authorized use.
4. FIS runs the query against the photo repository.
5. Visual comparison performed.
6. Background check conducted.
7. **Peer review** of results.
8. **Supervisor final review**.
9. FIS Possible Match Report issued.

### Report Requirements
- Must include notification that a match alone does **NOT** constitute probable cause.

### Prohibited Uses
- Use of images **outside** the photo repository is **PROHIBITED** unless approved by:
  - Chief of Detectives
  - Intelligence and Counterterrorism Bureau
  - Chief of Intelligence

### Privacy
- All records are subject to privacy and confidentiality restrictions.

> **Sergeant Focus:** Supervisors conduct the final review in the facial recognition process. Ensure all subordinates understand that a possible match is an investigative lead only and must never be presented as establishing probable cause.
