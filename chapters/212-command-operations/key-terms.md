# Section 212 — Key Terms and Numbers

## Critical Terms

| Term | Definition |
|------|-----------|
| **CPR System** | Centralized Personnel Resource — attendance and roll call tracking system |
| **Digital Activity Log** | Primary activity recording tool on Department smartphone; required for all uniformed MOS below captain rank |
| **Interrupted Patrol Log** | Desk officer log tracking time members spend in the stationhouse during tour |
| **Rosario Material** | Documents, notes, or logs potentially discoverable in criminal proceedings |
| **Reach Train** | Train brought in front of or behind a stalled train so passengers can walk through |
| **Rescue Train** | Train brought alongside for cross-platform transfer via emergency evacuation devices |
| **RTOCC** | Rapid Transit Operations Command Center — primary transit coordination hub |
| **Emergency Alarm Box** | Located 400-600 feet apart under blue lights; removes power on ALL tracks in vicinity |
| **MOME** | Mayor's Office of Media and Entertainment — issues press credentials |
| **DCPI** | Deputy Commissioner, Public Information — notified for all press incidents (24/7) |
| **ERPO** | Extreme Risk Protection Order — court order prohibiting firearm purchase/possession |
| **VGTOF** | Violent Gang and Terrorist Organization File within NCIC |
| **RTCC-FIS** | Real Time Crime Center — Facial Identification Section |
| **Probe Image** | Image of unidentified person submitted for facial recognition comparison |
| **Photo Repository** | Approved facial recognition database containing ONLY arrest and parole photographs |
| **Possible Match Candidate** | Investigative lead ONLY — does NOT establish probable cause |
| **AMBER Alert** | Child abduction alert for victims 17 or under in imminent danger of serious harm/death |
| **ALPS** | Automated License and Permit System — electronic pistol license verification |
| **Public Counter** | Scanner machine counter that MUST read zero before polls open |
| **Protective Counter** | Scanner counter that is NOT expected to be zero — do not confuse with public counter |
| **TRS** | Telecommunications Relay Service — dial 711 for deaf/hard of hearing communication |
| **4th Platoon Rule** | Members on 1800x0200 subpoenaed for next-day court reassigned to 3rd platoon (1600x2400) |

## Critical Numbers

| Number | Significance |
|--------|-------------|
| **1 hour** | Contact desk officer before end of tour (foot patrol needing relief); arrive at poll site before opening |
| **30 minutes** | Dispatch another unit if Transit call not finalized; radio check intervals during malfunction |
| **100 feet** | No-electioneering zone around poll site entrance |
| **400-600 feet** | Spacing of emergency alarm boxes (under blue lights) in subway |
| **2100 hours** | Poll closing time — persons on line may vote, no new entries |
| **72 hours** | HPD deadline to seal vacated premises |
| **60 days** | Rifle/shotgun surrender investigation completion deadline |
| **2 years** | Pistol license validity period; minimum firearm retention under ERPO |
| **8 hours** | Window to cancel AMBER Alert via fax to NYS Police |
| **30 days** | ERPO Unit reviews before expiration for renewal determination |
| **$15** | Subpoena fee — Civil/State |
| **$40** | Subpoena fee — Civil/Federal |
| **$5** | Medical records fee for outside parties |
| **711** | TRS dial code for deaf/hard of hearing relay service |
| **(212) 504-4115** | TTY number for 311 system |
| **0900-1700** | License Division hours (Mon-Fri); Legal Bureau ERPO Unit hours (Mon-Fri) |

## Handling Codes (VGTOF)

| Code | Action | Key Detail |
|------|--------|-----------|
| **1** | ARREST | Warrant exists — approach with caution |
| **2** | DETAIN | Reasonable suspicion — active investigation |
| **3** | INVESTIGATION | Do not alert — possible terrorism ties |
| **4** | INFORMATION GATHERING | Do not alert individual |
