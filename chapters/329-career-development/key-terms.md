# Section 329 — Key Terms and Definitions

## Retirement Definitions

| Term | Definition |
|------|-----------|
| **Terminal Leave** | Leave with pay computed at PPF; 1 calendar month per 10 years + 3 days per year beyond 20 + 1 day per 4 months |
| **LEOSA Certification Card** | Wallet-sized card exempting retired LEOs from local/state concealed carry prohibitions; must accompany retired ID card |
| **Vested Interest** | Discontinuance of police service with pension benefit eligibility but not full retirement |
| **Resignation** | Voluntary separation other than retirement/vested interest |
| **Dismissal Probation** | Period included in negotiated penalties for alcohol-related motor vehicle incidents |
| **NICS Check** | National Instant Criminal Background Check — required to retain post-1/15/13 Department-issued firearms |

## Key Forms

| Form | Purpose |
|------|---------|
| **PROPERTY RECEIPT - DISCONTINUANCE OF SERVICE (PD520-013)** | Property return documentation for retiring/resigning members |
| **PENSION APPLICATION (PD440-045)** | Grants terminal leave |
| **RESIGNATION AND EXIT INTERVIEW (PD452-151)** | Completed and signed by resigning member |
| **PROPERTY CLERK INVOICE (PD521-141)** | Lists firearms safeguarded |
| **ACQUISITION OR DISPOSITION OF FIREARMS (PD424-150)** | Report to NY State Police for each firearm |
| **HANDGUN LICENSE APPLICATION (PD643-041)** | Typed and notarized; for NYC residents applying for handgun license upon retirement |

## Key Timeframes

| Item | Timeframe |
|------|-----------|
| Tier 2 reporting to PPF | At least **30 calendar days** in advance |
| Tier 3 reporting to PPF | May report **without notice** |
| Terminal leave PPF reporting | Requested **10 days** prior (not required) |
| LEOSA card expiration | **12 months** from last range requalification |
| NICS check | Within **30 days** prior to separation |
| RESIGNATION AND EXIT INTERVIEW distribution | Within **5 business days** |
| Reinstatement after resignation | Only by **Police Commissioner** within **1 year** |
| Tier 2 mandatory retirement age | **63** |
| Tier 3 mandatory retirement age | **62** |
| Tier 3 retired ID/LEOSA issuance | **30 days** after discontinuance |
| Terminal leave discontinuance — vacation eligibility | Not for **30 days** after |

## Terminal Leave Restrictions

| Restriction |
|------------|
| No leaves of any kind after terminal leave begins |
| No sick report (unless LOD action while on terminal leave) |
| Excluded from excusal limitations |
| Available only for **emergency situations** |
| Cash overtime only if ordered to work |
| No programmatic overtime, fill details, or non-emergency duties (without bureau chief/DC approval) |

## Resignation Key Rules

| Rule |
|------|
| Bars reinstatement except by Police Commissioner within 1 year |
| Benefits/pension contributions NOT returned until ALL property accounted for |
| Police Commissioner decides if resignation is "without permission" |
| Firearms delivered to **borough Property Clerk's Office ONLY** |
| Report to PPF on last working day in **civilian clothes** |
| Sergeant and clerical member assigned to open locker of absent resignee |

## Memory Aids

> **Memory Aid — TERMINAL LEAVE MATH:** **1 month per 10 years** + **3 days per year beyond 20** + **1 day per 4 months** partial year. Example: 23 years = 2 months (20 years) + 9 days (3 × 3 years) = 2 months + 9 days terminal leave.

> **Memory Aid — TIER 2 vs TIER 3 RETIREMENT AGE:** **Tier 2 = 63rd birthday**, **Tier 3 = 62nd birthday.** Think: "**2** has **3** digits (63), **3** has **2** digits (62)" — or: Tier **2** gets the **higher** age (63), Tier **3** gets the **lower** age (62).

> **Memory Aid — TIER 2 vs TIER 3 PPF NOTICE:** **Tier 2 = 30 days advance notice** ("Tier Two = Takes Time, needs 30 days"). **Tier 3 = No notice required** ("Three = Free, no notice").

> **Memory Aid — RESIGNATION REINSTATEMENT:** Only **Police Commissioner** can reinstate, and only within **1 year.** Think: **PC** = **P**olice **C**ommissioner = **P**erson who has **C**omplete authority.

> **Memory Aid — FIREARMS UPON RESIGNATION:** Deliver to **borough Property Clerk's Office ONLY** ("Resigning = **B**orough **P**roperty **C**lerk, not Manhattan" — actually both Manhattan AND borough PC deliver, but the key exam point is borough PC is where resignations go, not the member's choice). LEOSA holders don't deliver firearms.
