# Section 218 — Invoicing Property: General Procedures

## P.G. 218-01: Invoicing Property — General Procedure

**Date Effective:** 06/26/25 | **Last Revision:** R.O. 53 | **Pages:** 4

### Purpose

To record and process property coming into police custody.

### Key Definitions

| Term | Definition |
|------|-----------|
| **Arrest Evidence** | Property seized for court presentation |
| **DNA Arrest Evidence** | Property seized for court presentation containing potentially probative DNA evidence |
| **Investigatory** | Property seized for investigation where a COMPLAINT REPORT and no ONLINE BOOKING SYSTEM ARREST WORKSHEET has been prepared |
| **DNA Investigatory** | Investigatory property containing potentially probative DNA evidence |
| **Forfeiture** | Arrest/investigatory evidence seized when probable cause existed to believe property was used to facilitate a crime or represents proceeds/substituted proceeds of a crime |
| **Decedent's Property** | Personal property of decedent |
| **Found Property** | Non-contraband property found by a member of the service or civilian |
| **Peddler's Property** | Items (e.g., stands, carts) removed from vendors |
| **Amnesty** | Firearm surrendered via Cash for Guns Program |
| **Safekeeping** | Property that is to be returned to owner |
| **Determine True Owner** | Property taken into custody where true owner cannot be determined and immediate arrest is not warranted |
| **Photo Release** | Recovered stolen vehicles where arrest was made and vehicle not used in any other crime |
| **Rotation Tow** | Non-evidence stolen or apparently abandoned vehicles including motorcycles recovered within NYC |

### Property Types

Firearms, Controlled Substances, Currency, Jewelry, Vehicles/Boats, General Property, Evidence Collection Kits/Swabs, Explosives.

### Procedure

#### Uniformed Member of the Service (Steps 1-8)

| Step | Action |
|------|--------|
| 1 | Enter circumstances and description of property in digital Activity Log |
| 2 | Notify desk officer |
| 2a | If vehicle unable to be driven, inform desk officer to request authorized tow/rotation tow |
| 3 | Prepare PROPERTY CLERK INVOICE WORKSHEET (PD521-141A) |
| 3a | Comply with FORMS Property instructions |
| 4 | Notify Stolen Property Inquiry Unit to cancel or transmit alarm |
| 5 | Notify owner, if known |
| 6 | Cancel alarm for stolen vehicle ONLY after owner takes possession or vehicle removed to Property Clerk/rotation tow facility |
| 7 | Enter notifications and inquiry results on WORKSHEET |
| 8 | Deliver property and completed WORKSHEET to desk officer of command of record |

#### Desk Officer (Steps 9-10)

| Step | Action |
|------|--------|
| 9 | Review WORKSHEET for accuracy |
| 10 | Submit approved WORKSHEET to command clerk for entry into FORMS Property |

#### Command Clerk (Steps 11-12)

| Step | Action |
|------|--------|
| 11 | Enter WORKSHEET into FORMS Property to generate PROPERTY CLERK INVOICE (PD521-141) |
| 12 | Record time and name of FINEST operator who transmitted alarm cancellation on appropriate INVOICE |

#### Uniformed Member (Steps 13-16)

| Step | Action |
|------|--------|
| 13 | Ensure all property is properly tagged, packaged or sealed, and clearly identified by INVOICE number and barcode |
| 14 | Ensure every bag/security envelope or item has a FORMS Property bar-coded label affixed to upper right hand corner |
| 14a | Affix bar-coded label to upper right hand corner of windshield for automobiles/boats |
| 15 | Digitally sign INVOICE verifying accuracy and completeness |
| 16 | Complete VEHICLE REPORT in FORMS if vehicle, bicycle, moped, scooter or e-bike is seized as result of vehicle stop |

#### Desk Officer (Steps 17-21)

| Step | Action |
|------|--------|
| 17 | Review and digitally sign INVOICE |
| 18 | Photocopy INVOICE and forward to CO, Firearms and Tactics Section if item is a bullet resistant garment |
| 19 | Ensure property is properly stored and location entered into FORMS Property |
| 20 | Safeguard property until properly disposed of |
| 21 | Telephone NYS Tax Office for pickup of property owned by Division of Lottery — DO NOT send to Property Clerk |

### Additional Data — Separate Invoices Required

A separate PROPERTY CLERK INVOICE will be prepared for each of:

- Different property types
- Different property categories
- Different owners
- Property delivered to different locations (Police Laboratory, Property Clerk, Bomb Squad, OCME, etc.)

> **Exam Alert:** When items will be sent to Police Laboratory, the INVOICE will **not exceed twenty line items**.

> **Exam Alert:** Do NOT use terms "pairs" or "sets" when invoicing — enter the actual number of items in the "quantity" box.

> **Exam Alert:** When a vehicle has obvious fire damage, notify FDNY borough dispatcher but do NOT prepare a COMPLAINT REPORT — a Fire Marshal will determine if arson occurred.

## P.G. 218-02: Return of Property/Vehicles at Command and Processing Voided Property Invoices

**Date Effective:** 06/02/25 | **Last Revision:** R.O. 37 | **Pages:** 4

### Purpose

To return all property listed on an invoice to legal claimant at a command and process voided PROPERTY CLERK INVOICES.

### Procedure — Return of Property

#### Desk Officer (Steps 1-7)

| Step | Action |
|------|--------|
| 1 | Require claimant to produce valid identification, preferably photo ID |
| 1a | Upload copy of claimant's ID digitally to INVOICE using "Add Remarks/Document" function |
| 2 | Check property/vehicle against INVOICE |
| 3 | Request claimant examine security seal and check against number on INVOICE prior to breaking seal |
| 4 | Complete "Release/Disposition — Return To Owner" function using FORMS Property |
| 4a | Have claimant sign using digital signature capture device |
| 5 | Ensure alarm has been canceled |
| 6 | Approve delivery of property utilizing username and password |
| 7 | Release property/vehicle to claimant |

> **Exam Alert:** When no physical photo ID is available, a **photograph of the claimant** will be taken with a Department smartphone and digitally attached to the INVOICE.

### Investigatory Evidence Return

| Step | Action |
|------|--------|
| 9 | Conduct investigation to determine if property must be retained |
| 9a | Confer with appropriate investigative unit |
| 9b | Authorize release **no more than ten days** after receipt of request, if no longer required as evidence |
| 10 | Inform claimant property may not be released if still required for specific criminal proceeding or active investigation |
| 11 | Designate member to prepare OFFICIAL LETTERHEAD with INVOICE number(s) and reasoning for denial |

> **Exam Alert:** Investigatory evidence: desk officer must authorize release or denial within **10 days**. If denied, claimant may request CO/designee of investigative unit to review within **5 days**.

### Voiding Invoices

| Step | Action |
|------|--------|
| 14 | Use "Void Invoice" function if INVOICE has NOT been approved by a supervisor and cannot be corrected |
| 15–16 | If INVOICE has been approved by supervisor: create NEW invoice, then void the old one with cross-references |

> **Exam Alert:** Once approved by a supervisor, an INVOICE can **only be voided by a supervisor** assigned to the command where property is physically located.

> **Exam Alert:** Filed INVOICES maintained for **six years** — each July, previous year's invoices boxed and forwarded to Property Clerk warehouse.

## P.G. 218-03: Delivery of Property Other Than Vehicle/Boat to Property Clerk

**Date Effective:** 12/31/24 | **Last Revision:** R.O. 78 | **Pages:** 2

### Purpose

To forward invoiced property to Property Clerk.

### Key Points

- Desk officer creates PROPERTY TRANSFER REPORT (PD521-1412) using FORMS Property
- Command messenger compares INVOICES against property for completeness
- If discrepancies exist: immediate investigation, notify CO/duty captain and **Internal Affairs Bureau (IAB) Command Center**
- IAB Log Number entered in "Remarks" section of INVOICE
- Property Clerk member confirms acceptance in FORMS Property
- **Integrity Control Officer** verifies all property not forwarded is accounted for and secured

> **Sergeant Focus:** Any discrepancy in property against INVOICE requires immediate IAB notification — this is a critical integrity safeguard.

## P.G. 218-05: Recording of Arrest and Property Clerk Invoice Numbers on Arrest Related Reports

**Date Effective:** 12/31/24 | **Last Revision:** R.O. 78 | **Pages:** 2

### Purpose

To determine disposition of prisoners and allow processing/disposing of property invoiced as arrest evidence.

### Key Points

- Invoicing member must enter INVOICE number on ON-LINE BOOKING SYSTEM ARREST WORKSHEET (PD244-159)
- Prepare REQUEST FOR LABORATORY EXAMINATION REPORT (PD521-168) if necessary
- **All defendants' names must be listed on the REQUEST** — if not, evidence could be destroyed before accomplices/co-defendants are tried, resulting in dismissed charges

> **Exam Alert:** Failure to list all defendant information on the REQUEST FOR LABORATORY EXAMINATION REPORT could lead to **premature destruction of evidence** and dismissal of criminal charges against co-defendants.

## P.G. 218-06: Temporary Removal of Invoiced Property from the Command

**Date Effective:** 05/20/24 | **Last Revision:** R.O. 31 | **Pages:** 1

### Purpose

To account for invoiced property temporarily removed from the command to court or other authorized agency.

### Key Points

- Desk officer verifies identity of requesting member via IDENTIFICATION CARD (PD416-091) if from another command
- PROPERTY TRANSFER REPORT prepared and digitally signed by both requesting officer and desk officer
- Member obtains receipt if property retained by court or other agency
- Receipt forwarded to Property Clerk Borough Office for FORMS Property entry
- All official requests for property should be accompanied by a **subpoena** from the requesting agency (except ADAs and assistant corporation counsels)

> **Sergeant Focus:** The desk officer must verify identity, ensure proper documentation, and make Command Log entries for every temporary property removal.

| REQUEST FOR LABORATORY EXAMINATION REPORT | PD521-168 |
| OFFICIAL LETTERHEAD | PD158-151 |
| IDENTIFICATION CARD | PD416-091 |
