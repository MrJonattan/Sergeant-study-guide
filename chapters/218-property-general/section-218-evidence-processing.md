# Section 218 — Evidence Processing: Specialized Items

## P.G. 218-11: Use of Biohazard Labels

**Date Effective:** 05/20/24 | **Last Revision:** R.O. 31 | **Pages:** 1

### Purpose

To identify regulated waste coming into police custody.

### Definition

**REGULATED WASTE** — Liquid or semi-liquid blood or other potentially infectious materials; contaminated items that would release blood/infectious materials if compressed; items caked with dried blood capable of releasing during handling; contaminated sharps; pathological and microbiological wastes containing blood or other potentially infectious materials.

### Procedure

| Step | Role | Action |
|------|------|--------|
| 1 | Member | Follow normal invoicing procedures |
| 2 | Member | Request Biohazard Label(s) from desk officer |
| 3 | Member | Affix label as close as feasible using string, wire, adhesive or other method preventing loss/removal |
| 4 | Member | Use one label for each item or container in same case |
| 5 | Desk Officer | Request additional Biohazard Labels from Quartermaster Section when needed |

> **Exam Alert:** Biohazard Labels are **fluorescent orange** with the word "Biohazard" on the label's face.

## P.G. 218-23: Processing Firearms and Firearm-Related Evidence

**Date Effective:** 12/31/24 | **Last Revision:** R.O. 78 | **Pages:** 11

### Purpose

To ensure all firearms and firearm-related evidence is properly marked, packaged, sealed, and invoiced to maintain a continuous chain of custody.

### Key Definitions

| Term | Definition |
|------|-----------|
| **Firearm** | Rifle, shotgun, pistol, revolver, derringer, machine gun; starter's pistol, zip gun, air gun, CO2 gun; homemade/modified gun; simulated/toy/imitation firearm; non-metal-detectable firearm; ghost gun/untraceable; 3D-printed firearm; weapon with housing designed to expel a projectile by explosive action (including stabilizing/arm brace modifications) |
| **Cartridge** | Live ammunition — bullet (projectile) + shell casing as one unit |
| **Fired Bullet** | Part of cartridge fired through barrel (lead, metal-jacketed, coated lead) |
| **Shell Casing** | Metal casing remaining in/ejecting from firearm after bullet leaves barrel |

### Cartridge Categories (Critical for Packaging)

| Category | Description | Packaging Rule |
|----------|-------------|---------------|
| a | Cartridge removed from **chamber** | Package and seal in ONE container; write "Cartridge Removed From Chamber" |
| b | Remaining cartridges removed from firearm (magazine/cylinder) | ALL in ONE container; write "Cartridge(s) Removed From Firearm" |
| c | Cartridges NOT from firearm, seized from person/location, NO crime scene | ONE container per person; ONE container per location; identify person/location |
| d | Cartridges NOT from firearm, recovered from **crime scene** | EACH cartridge in SEPARATE container |

### Marking Rules

| Item | Where to Mark | Do NOT Mark |
|------|--------------|-------------|
| Cartridges | Bullet portion | Metal shell casing portion |
| Fired Bullets | Bottom/base | Sides (near rifling marks) |
| Shell Casings | Inside | Exterior or bottom |
| Firearms | Initials + unique consecutive number | — |
| Magazines | Initials + unique consecutive number | — |
| Silencers | Initials + unique consecutive number | — |

> **Exam Alert:** Use a **scribe or sharp writing instrument** to scratch initials and a unique consecutive number on each item. Each number must be used only once per case.

### Crime Scene Evidence — Individual Packaging Rules

| Evidence Type | Packaging |
|--------------|-----------|
| Cartridges from crime scene | EACH in SEPARATE container — NEVER two or more together |
| Fired bullets from crime scene | EACH in SEPARATE container — NEVER two or more together |
| Shell casings from crime scene | EACH in SEPARATE container — NEVER two or more together |
| Cartridges from person (no crime scene) | ALL from same person in ONE container |
| Cartridges from location (no crime scene) | ALL from same location in ONE container |

### Sealing Containers

| Method | Procedure |
|--------|-----------|
| Standard container | Fasten with EVIDENCE TAPE; sign name across border between tape and container |
| NYPD Plastic Security Envelope | Remove paper strip exposing pre-applied adhesive; fold top aligning blue circles; sign on pre-printed line |

### WORKSHEET Listing Order

1. Firearm (item #1) — describe by make, type, serial number
2. Magazine (if any)
3. Cartridge from chamber
4. Remaining cartridges from firearm (same caliber = one item)
5. Silencer(s) — each listed separately
6. Firearm accessories affixed to firearm or needed for operation
7. Additional firearms and related items (repeat sequence)
8. Cartridges not from firearm (per person/per location)
9. Shell casings from crime scene (same caliber = one item)
10. Fired bullets from crime scene (all = one item)

### Desk Officer Responsibilities

| Step | Action |
|------|--------|
| 21 | Verify firearm is unloaded (if unfamiliar, notify ESU to respond) |
| 21b | Verify proper marking, packaging, and sealing |
| 21c | Verify INVOICES and forms properly prepared |
| 22 | Enter notation in "Remarks" that firearm(s) have been unloaded/rendered safe |
| 23 | Review COMPLAINT REPORT — verify address, perpetrator info, evidence section, weapon section |
| 26 | Forward firearms requiring lab analysis to Police Laboratory |

> **Exam Alert:** Firearms requiring lab analysis must be delivered to the **patrol borough office by 0100 hours daily**. Delay requires **exigent circumstances** AND permission from **precinct CO/duty captain**.

> **Exam Alert:** Firearm alarm cancellation is made ONLY by the **Stolen Property Inquiry Unit** — recovering command does NOT cancel.

### Firearms Invoiced for Safekeeping

Delivered DIRECT to Property Clerk Division (no lab analysis required):
- Property of retired, resigned, dismissed, suspended, sick, or hospitalized members
- Lawfully possessed property of a deceased person
- Obtained from licensed dealer when license revoked/suspended/expired
- Lawfully possessed and removed by court order/order of protection
- Obtained from permit holder during renewal/expiration/cancellation

### DNA Evidence on Firearms

- DO NOT package DNA evidence if **wet** — must be **air-dried** first
- DO NOT use plastic bags/envelopes for DNA evidence — use **paper bags/containers**
- Affix orange Biohazard label to each paper bag/container
- If marking would contaminate DNA evidence, mark the container instead (print name in BLOCK letters)

> **Exam Alert:** Firearm accessories NOT affixed to the firearm AND not needed for operation (holster, gun case, sling) are invoiced on a **separate INVOICE** from the firearm.

## P.G. 218-31: Processing Computer Evidence

**Date Effective:** 05/20/24 | **Last Revision:** R.O. 31 | **Pages:** 4

### Purpose

To process and safeguard computer-related evidence coming into Department custody.

### Key Definitions

| Term | Definition |
|------|-----------|
| **CPU** | Box-shaped unit housing internal hard drive, memory, and central processor |
| **Monitor** | Television-type display instrument |
| **Keyboard** | Input device resembling a typewriter |
| **Mouse** | Pointing device connected to CPU by cable |
| **Printer** | Output device (dot matrix, ink-jet, laser) |
| **Peripherals** | Scanners, tape drives, modems, speakers, personal digital assistants, pocket organizers, laptops |

### Notifications

| Time | Contact |
|------|---------|
| Mon–Fri, 0600–2000 | Computer Crimes Squad |
| All other times | Office of the Chief of Detectives |

### Marking and Packaging

| Step | Action |
|------|--------|
| 3a | Affix identifiable mark to hardware (monitor, CPU, printer, etc.) |
| 3b | Label items to indicate which peripherals were connected (Monitor 1, CPU 1, Keyboard 1) and tag all wires |
| 3c | Package similar floppy disks in **paper envelopes** — do NOT remove disks from drives; secure with masking tape |

> **Exam Alert:** DO NOT use Plastic Security Envelopes for floppy disks — **static electricity** can destroy data. DO NOT write on floppy disks with **ballpoint pens** — use labels or felt tip markers.

### Critical Prohibitions

- Members will **NOT search** hard drives, floppy disks, CD-ROMs, or data storage devices — Computer Crimes Squad ONLY
- Searching may **inadvertently alter or destroy** evidence
- Do not touch keyboard or execute programs at the scene
- If modem attached, **unplug telephone cord** from wall jack immediately
- To power off: **unplug from wall outlet** — do NOT use on/off switch; photograph monitor screen first
- Do NOT transport in vehicle trunks with radio transceivers
- Do NOT expose to devices emitting magnetic fields (portable radios, car speakers)

## P.G. 218-33: Processing Sexual Offense Collection Kits

**Date Effective:** 06/19/25 | **Last Revision:** R.O. 42 | **Pages:** 4

### Purpose

To ensure sexual offense evidence collection kits and other evidence are properly forwarded to the OCME Evidence Unit or Property Clerk.

### Key Definitions

| Term | Definition |
|------|-----------|
| **Sexual Offense** | Offenses enumerated in Article 130, NYS Penal Law |
| **Sexual Offense Evidence Collection Kit** | Kit used by trained medical personnel in SAFE Program hospitals; contains slides, swabs, test tubes, envelopes |
| **Drug Facilitated Sexual Assault Kit** | Separate kit with two gray-topped blood tubes and urine container; used when drugs/alcohol suspected |
| **Sexual Offense Evidence** | Items containing bloodstains, serology, body fluids, biological evidence (clothing, bedding, undergarments) for DNA profiling |

### Procedure

| Step | Role | Action |
|------|------|--------|
| 1 | Member | Ensure victim transferred to SAFE Program hospital |
| 2 | Member | Take possession of kits and evidence upon completion of medical exam (with victim consent) |
| 3 | Member | Ensure physician's and complainant's names entered on kit |
| 4 | Member | Remove evidence to precinct of occurrence; prepare WORKSHEET |
| 5–8 | Desk Officer | Direct preparation of INVOICE, REQUEST FOR LAB EXAM, COMPLAINT REPORT |
| 9 | Desk Officer | Direct delivery of kit and COMPLAINT REPORT copy to Police Laboratory |
| 10 | Desk Officer | Direct delivery of OTHER sexual offense evidence to Property Clerk (if kit was used at hospital) |
| 11 | Desk Officer | If NO kit used at hospital, deliver other evidence to Police Laboratory |
| 12 | Desk Officer | Comply with P.G. 218-49 for "high priority" cases |
| 14 | Desk Officer | Immediately notify NYPD OCME Liaison Unit of all unfounded cases |

> **Exam Alert:** THE SEAL on a Sexual Offense Evidence Collection Kit **SHALL NOT BE BROKEN** by any member of the Department EXCEPT authorized personnel.

> **Exam Alert:** Separate INVOICES required: one for the Sexual Offense Evidence Collection Kit, one for the Drug Facilitated Sexual Assault Kit, and one for other sexual offense evidence.

> **Exam Alert:** A member of the service (male or female) is NOT required to be present during the medical examination. **Chain of custody** during examination is the sole responsibility of attending medical personnel.

> **Exam Alert:** Sexual Offense Evidence Collection Kit should be used when the offense occurred within **120 hours** or at the discretion of medical personnel.

## P.G. 218-49: Immediate Delivery of DNA Evidence to Police Laboratory and OCME

**Date Effective:** 05/20/24 | **Last Revision:** R.O. 31 | **Pages:** 4

### Purpose

To provide a standard procedure for the immediate delivery of DNA evidence from the invoicing command direct to the Police Laboratory, and subsequently, immediately and directly to the OCME.

### Delivery Chain

```
Command → Police Laboratory → OCME Evidence Unit
```

### Detective Supervisor Responsibilities

| Step | Action |
|------|--------|
| 1 | Determine if circumstances require immediate delivery based on: conferral with assigned investigator, CSU/ECT personnel, type of crime, nature of perpetrator, results of investigation |
| 2 | Confer with FIU OCME Liaison Unit personnel and Detective Borough DNA Coordinator |
| 3 | Confer with desk officer; request uniformed member for immediate delivery |
| 4 | Follow up to ensure DNA evidence was immediately delivered through the full chain |

### Police Laboratory Evidence Control Supervisor

| Step | Action |
|------|--------|
| 11 | Inspect DNA evidence — verify proper packaging, sealing, INVOICE listing, forms |
| 11f | Confirm DNA evidence is probative and requires immediate delivery to OCME |
| 12 | Assign "Police Laboratory Number" to each INVOICE |

### OCME Evidence Unit

| Step | Action |
|------|--------|
| 21 | Perform intake using FORMS Property |
| 22 | Confirm acceptance of PROPERTY TRANSFER REPORT |
| 23 | Print five copies of PROPERTY TRANSFER REPORT ACCEPTANCE DOCUMENT — three to messenger, two retained |

> **Exam Alert:** The delivering member must telephone the Police Laboratory Evidence Control Section after OCME acceptance and provide the OCME "Evidence Control Number" for each INVOICE.

> **Sergeant Focus:** Detective supervisors must ensure ECMS entry documenting date/time of delivery, Police Laboratory Number, and OCME Evidence Control Number for all DNA evidence.

## P.G. 218-51: Suspect Evidence Collection Kits

**Date Effective:** 06/19/25 | **Last Revision:** R.O. 42 | **Pages:** 3

### Purpose

To ensure Suspect Evidence Collection Kits are properly collected and processed.

### Definition

**SUSPECT EVIDENCE COLLECTION KIT** — A kit used by medical personnel in hospitals to gather probative forensic evidence from a suspect or arrestee.

### Key Procedure Steps

| Step | Role | Action |
|------|------|--------|
| 1 | Assigned Investigator | Determine if suspect has probative forensic evidence on person/clothing |
| 2 | Assigned Investigator | Take measures to protect forensic evidence from alteration, loss, contamination |
| 3 | Assigned Investigator | Obtain voluntary consent (CONSENT TO SUSPECT EVIDENCE COLLECTION KIT, PD321-031) |
| 3c | Assigned Investigator | If no consent, contact **Legal Bureau** for possible court order through DA's Office |
| 4 | Assigned Investigator | Escort suspect to a designated hospital — DO NOT use same hospital as victim |
| 5 | Assigned Investigator | Alert HHC hospital and SART before bringing suspect |
| 7 | Assigned Investigator | Ensure investigator of **same sex** as suspect remains during examination |
| 9 | Assigned Investigator | DO NOT collect separate DNA Exemplar Kit when exemplar already collected as part of Suspect Evidence Collection Kit |
| 14 | Assigned Investigator | Prepare SEPARATE INVOICE for each Suspect Evidence Collection Kit — do NOT invoice other evidence on same INVOICE |

> **Exam Alert:** Neither consent nor a court order is necessary to **invoice a suspect's clothing** (including underwear) or **document/photograph** a suspect's physical features (bite marks, tattoos, etc.).

> **Sergeant Focus:** Detective supervisors must ensure Biohazard Labels are properly utilized, review each INVOICE and REQUEST, and ensure ECMS entries documenting delivery are prepared.

## P.G. 218-53: Processing Fireworks Evidence

**Date Effective:** 05/20/24 | **Last Revision:** R.O. 31 | **Pages:** 5

### Purpose

To establish a standard procedure to invoice fireworks coming into custody.

### Key Definitions

| Term | Definition |
|------|-----------|
| **Fireworks Evidence** | Bags, boxes, and/or loose items of fireworks |
| **Heavy Duty Bag (For Fireworks)** | Specific Department-issued clear vinyl bag for collection/storage of loose fireworks |

### Invoicing Procedure

| Step | Action |
|------|--------|
| 1 | Take two sets of photographs (one for ADA, one for arrest evidence) |
| 3 | Create INVOICE for photographs as arrest evidence |
| 4 | Create **separate** INVOICE for fireworks evidence |
| 7 | Place all loose fireworks into Department-issued Heavy Duty Bags; close with specific seals |
| 8 | Ensure bag is not overfilled |
| 9 | Separate "party snaps," "cracker balls," "torpedoes" from other fireworks (chain reaction risk) |
| 10 | Do NOT reopen bags once sealed |
| 11–13 | Enter quantity of sealed bags/boxes — do NOT individually count fireworks |
| 14 | Attach photocopy of INVOICE to each sealed bag/box (minimum 10 for large seizures) |

### Desk Officer — Delivery to Bomb Squad

| Step | Action |
|------|--------|
| 19 | Contact **Bomb Squad Range (Rodman's Neck)** Mon–Fri, 0700–1500 |
| 19a | Contact Bomb Squad Office at all other times |
| 22 | Create PROPERTY TRANSFER REPORT using FORMS Property |
| 23 | Make Command Log entry (INVOICE numbers, delivering member, vehicle numbers) |

### Transportation Safety Rules

| Rule | Details |
|------|---------|
| No smoking | Within **50 feet** of any fireworks |
| RMP transport | Maximum **4 bags** in rear seat |
| Passenger van | Maximum **4 bags per seat** on rear bench |
| Cargo van | Maximum **25 sealed bags/boxes** |
| Larger seizures (>25) | Closed-body truck with separate cab and wood cargo floor; secure load |
| Windows | Open minimum **2 inches** to avoid toxic fumes |
| Trunk storage | **Prohibited** |
| After dark | **No unloading** at Bomb Squad range |
| Significant amount | Notify FDNY through **311** for fire protection escort |

> **Exam Alert:** ONLY Department-issued "Heavy Duty Bags (For Fireworks)" may be used — peddler, garbage, or recycling bags are **strictly prohibited**.

## P.G. 218-54: Preparation of Request for Laboratory Examination Report (PD521-168)

**Date Effective:** 05/20/24 | **Last Revision:** R.O. 31 | **Pages:** 1

### Purpose

To provide guidelines for proper preparation of the REQUEST FOR LABORATORY EXAMINATION REPORT using FORMS Property.

### Procedure

| Step | Role | Action |
|------|------|--------|
| 1 | Invoicing Officer | Select "YES" to "Is a Request for Laboratory Examination required?" when preparing INVOICE |
| 2 | Invoicing Officer | Select evidence type: DNA, Firearms/Firearms Related, Criminalistics, or Other |
| 3 | Invoicing Officer | Ensure DNA ELIMINATION SAMPLES section is completed |
| 4 | Invoicing Officer | Select and add all Analysis Exam Numbers for each item on INVOICE |
| 5 | Invoicing Officer | Digitally sign REQUEST verifying accuracy and completeness |
| 6 | Desk Officer | Digitally sign REQUEST verifying accuracy and completeness |

> **Exam Alert:** A REQUEST FOR LABORATORY EXAMINATION REPORT must be prepared for **each INVOICE** that lists evidence submitted to the laboratory for analysis.
