# Section 219 — Key Terms, Timeframes, and Forms

## Key Definitions

| Term | Definition |
|------|-----------|
| **Category I Vehicle** | Unmarked vehicle not required to be stored at Department facility; permitted to be removed from the City regularly; designated by the Police Commissioner |
| **Category II Vehicle** | Unmarked vehicle with specific designations (exempt from OPP garage parking ineligibility) |
| **General Purpose Vehicle** | Standard unmarked vehicle for routine operational use |
| **Special Purpose Vehicle** | Unmarked vehicle assigned for specific operational needs |
| **License Plate Reader (LPR)** | Specialized camera/computer that captures license plate numbers, converts to text, compares against "hot list"; NOT connected to NYSPIN; activates on partial matches |
| **Mobile Digital Device** | Any smartphone, tablet, or portable computer issued by the Department |
| **Vehicle Identification Box (VIB)** | In-vehicle transponder for automated fuel dispensing |
| **NYPD Restricted Parking Permit (Misc. 23-N)** | Cardboard parking credential (~4"×10") with serial number, command code, license plate number, and expiration date |
| **NYCT NYPD OMNY Police Pass** | Transit pass for subway/bus access (not express bus); valid ~7 years; assigned to individual member by unique serial number |
| **Radiation Detection Pager** | Device assigned to commands or individual members for detecting radiological threats; requires specific training |
| **Ullage** | Remaining capacity in a fuel tank; tanks filled to 90% ullage |
| **Regulated Infectious Waste** | Biohazard materials picked up by Medical Division messenger |
| **OLBS** | Online Booking System — one of the systems ICO issues passwords for through Omniform |
| **BADS** | Booking and Arraignment Desk System — ICO issues passwords through Omniform |
| **LAPS** | LAPS system — ICO issues passwords through Omniform |
| **VIB** | Vehicle Identification Box — in-vehicle transponder for automated fuel dispensing |
| **CRALO** | Command Radio Auto Liaison Officer — holder of yellow fuel card |
| **ERS** | Employee Resources Section — where member reports for replacement shield/ID after loss |
| **SPIU / Stolen Property Inquiry Unit** | Unit notified to transmit alarm for lost/stolen Department property with identifying marks |
| **ITB** | Information Technology Bureau — provides Mobility Tablet tech support 24/7; handles LPR repairs; telecommunications unit handles lost/stolen mobile devices |
| **COG** | "Change of Gear" — the Digital Activity Log entry documenting vehicle inspection results (condition, odometer, gas gauge) |
| **DAS** | Domain Awareness System — ICO audits 5 random members monthly for compliance |
| **Focused Audit** | An audit of computer activity during a specific time period, requested by ICO or C.O. via 49 to Deputy Commissioner of IT |

## Critical Timeframes

| Timeframe | Rule | Source |
|-----------|------|--------|
| **Each tour** | Inspect Department vehicle; account for radiation detection pagers; account for LPR devices | 219-01, 219-27, 219-31 |
| **Each Sunday** | Weekly vehicle inspection after second platoon posted | 219-02 |
| **10th of month** | Car wash expense reports due to Audits and Accounts | 219-04 |
| **Nov 1 – Mar 15** | Tire chains available | 219-08 |
| **>4 inches snow** | Threshold for installing tire chains | 219-08 |
| **24 hours** | LPR hot list must be updated within this period to use device | 219-31 |
| **5 business days** | Request ITB permission before taking mobile device to foreign country | 219-32 |
| **30 days** | Notify CO of vehicle change for Restricted Parking Permit; max time to use substitute vehicle | 219-29 |
| **7 years** | General validity period for NYCT NYPD OMNY Police Pass | 219-26 |
| **Monthly** | ICO audits DAS usage for 5 random members | 219-14 |
| **Quarterly** | CO surveys generator/mechanical/electrical rooms | 219-28 |
| **50%** | Minimum fuel tank level for commands with single tank (heating + generator) | 219-28 |
| **90%** | Maximum fill level (ullage) for fuel deliveries | 219-12 |
| **12 months** | Period for counting red light (5+) or school speed (15+) camera violations for permit ineligibility | 219-29 |

## Lost/Stolen Firearm — County Notification Rules

| Location of Loss | Who to Notify First |
|-----------------|-------------------|

| Nassau or Suffolk County | Desk Officer **105 Pct** |
| Westchester, Putnam, Rockland, or Orange | Desk Officer **50 Pct** |
| Outside residence counties | **Operations Unit DIRECT** |
