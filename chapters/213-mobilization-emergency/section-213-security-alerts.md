# Citywide Security Alert Levels

## Overview

This section covers the NYPD's tiered security alert system, which mirrors the national threat advisory framework. Sergeants must know the five alert levels by color and name, the current default level for New York City, and how changes are communicated throughout the Department.

---

## P.G. 213-18 — Citywide Security Alert Levels

### Key Points

#### Five Alert Levels (Low to High)

| Level | Color | Name | Risk Description |
|-------|-------|------|-----------------|
| 1 | **GREEN** | Low | Low risk of terrorist attack |
| 2 | **BLUE** | Guarded | General risk of terrorist attack |
| 3 | **YELLOW** | Elevated | Significant risk of terrorist attack |
| 4 | **ORANGE** | High | High risk of terrorist attack |
| 5 | **RED** | Severe | Severe risk of terrorist attack |

#### Current Status
- New York City is currently maintained at **ORANGE (High)**

#### Communication
- Changes in alert level are communicated via **FINEST Message**
- Each alert level has corresponding **security posture adjustments**

### Supervisor Responsibilities
- Know the current alert level and its implications for patrol operations
- Monitor FINEST Messages for any changes to the citywide alert level
- Brief subordinates on any alert level changes and the corresponding security posture adjustments
- Adjust patrol activities and vigilance in accordance with the current level

> **Exam Alert:** NYC is maintained at **ORANGE (High)** -- the fourth of five levels. This is the default, not an escalated condition.

> **Exam Alert:** Alert level changes are communicated via **FINEST Message** -- not roll call, not Operations Order, but FINEST Message specifically.

> **Exam Alert:** The five levels in order from lowest to highest: **GREEN (Low) -> BLUE (Guarded) -> YELLOW (Elevated) -> ORANGE (High) -> RED (Severe)**.

> **Sergeant Focus:** Be prepared for exam questions that test the order of alert levels or ask which color corresponds to which threat description. A common distractor is confusing BLUE (Guarded/General risk) with GREEN (Low risk).

## Memory Aid — Alert Level Order

**G-B-Y-O-R** (Green, Blue, Yellow, Orange, Red)

Think: "**G**o **B**efore **Y**ou **O**verreact **R**ashly" -- levels escalate from calm (Green) to crisis (Red).

| Color | One-Word Risk | Mnemonic |
|-------|--------------|----------|
| GREEN | Low | Go |
| BLUE | General | Before |
| YELLOW | Significant | You |
| ORANGE | High | Overreact |
| RED | Severe | Rashly |
