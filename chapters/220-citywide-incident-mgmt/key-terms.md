# Section 220 — Key Terms and Definitions

## CIMS Organizational Terms

| Term | Definition |
|------|-----------|
| **CIMS** | Citywide Incident Management System — NYC's adaptation of NIMS/ICS |
| **Command Element** | Individual(s) responsible for overall command function; may be Single or Unified |
| **Single Command** | One Primary Agency with one Incident Commander |
| **Unified Command** | Multiple Primary Agencies, each with an Agency Incident Commander; all co-locate at one ICP |
| **Primary Agency** | Agency designated by the Primary Agency Matrix to command the incident |
| **Supporting Agency** | Agency assisting but not commanding |
| **Coordinating Agency** | NYCEM — facilitates inter-agency coordination and support |
| **Potential Primary Agency** | State/federal agencies with jurisdictional authority under specific conditions |
| **Subject Matter Expert Agency** | Agency with technical expertise but no tactical personnel deployed |
| **Core Competency** | Functional area of expertise assigned to one single agency |

## Command Staff Positions

| Position | Designation Authority (Override) |
|----------|-------------------------------|
| **Public Information Officer** | Mayor's Press Office may designate |
| **Safety Officer** | Required by law at HazMat incidents |
| **Liaison Officer** | NYCEM may designate |

## General Staff Sections

| Section | Chief Manages | Subordinate Elements |
|---------|-------------|---------------------|
| **Operations** | All tactical operations | Branches, Sectors (geographic), Groups (functional) |
| **Planning** | Information collection, IAP preparation, resource tracking | Resources Unit, Situation Unit, Demobilization Unit, Documentation Unit |
| **Logistics** | All service and support | Service Branch (Communications, Medical, Food), Support Branch (Supply, Facilities, Ground Support) |
| **Finance/Administration** | Financial/administrative functions | Time Unit, Procurement Unit, Compensation/Claims Unit, Cost Unit |
| **Intelligence/Investigations** | Investigative/intelligence operations | Intelligence Group, Surveillance/Evidence Group, Forensic Group, Missing/Unidentified Persons Group, Investigative Support Group |

## Incident Facilities

| Facility | Key Rule |
|----------|----------|
| **Incident Command Post (ICP)** | ONE per incident (except nuclear); identified by incident name |
| **Operations Section Post** | NOT co-located with ICP |
| **I/I Section Post** | NOT co-located with ICP or Operations Post |
| **Staging Area** | Resources in "Available" status; deploy within 3 minutes; named by geography/number |
| **Incident Base** | NOT activated in NYC (absent extraordinary circumstances) |
| **Camp** | NOT activated in NYC (absent extraordinary circumstances) |

## Resource Terms

| Term | Definition |
|------|-----------|
| **Single Resource** | One vehicle + crew, single personnel, K-9 team, or pre-designated team |
| **Resource Team** | Same Kinds/Types assembled for a tactical assignment; requires Leader + communications |
| **Special Response Team** | Different Kinds/Types assembled for a tactical assignment; requires Leader + communications |
| **Kind** | Broad class of similar resources |
| **Type** | Level of capability: I (greatest) to IV (least) |
| **Squad** | 1 Sergeant + 8 Police Officers |
| **Mobile Field Force** | 2 to 6 Squads |

## Resource Status Conditions

| Status | Description | Readiness |
|--------|-------------|-----------|
| **Assigned** | Performing work tasks under supervisor | Working |
| **Available** | Ready for assignment | Deploy within **3 minutes** |
| **Out-of-Service** | Not able to be assigned | Mechanical, rest, illness, injury |

## Incident Management Priorities

| Priority | Operations |
|----------|-----------|
| **#1** | Life Safety Operations (immediate, supersedes all) |
| #2 | Site Management Operations |
| #3 | Investigative Operations |
| #4 | Recovery/Restoration Operations |

## Span of Control

| Context | Ratio |
|---------|-------|
| General | **1:3 to 1:7** |
| Law Enforcement | **1:8 to 1:10** |

## Key Principles

| Principle | Definition |
|-----------|-----------|
| **Span of Control** | Number of personnel one supervisor can effectively manage |
| **Unity of Command** | Every subordinate has only ONE designated supervisor |
| **Chain of Command** | Direction given through established authority levels |
| **Management by Objectives** | Incident Objectives must be Attainable, Measurable, Adaptable/Flexible |

## Critical Timeframes and Thresholds

| Item | Threshold |
|------|-----------|
| Operational Period | Usually not over **24 hours** |
| Written IAP required | When incident extends beyond **24 hours** (multiple Operational Periods) |
| Available resource readiness | **3 minutes** to deploy |
| Check-In | Required **before** assignment to Operations |

## CBRN/HazMat Command Rules

| Situation | Command Structure |
|-----------|------------------|
| Actual/suspected crime or terrorism | NYPD **Single Command** (Incident Commander) |
| No crime/terrorism | **Unified Command** (NYPD, FDNY, DEP, DOHMH) |
| Nuclear weapon (anticipated/suspected/actual) | **Unified Command** with borough-centric ICPs |

## IMT Activation Authority

| IMT Type | Activated By |
|----------|-------------|
| Citywide | Police Commissioner, First Deputy Commissioner, or Chief of Department |
| Borough | Chief of Patrol or Patrol Borough Commander (or Duty Chief) |
| Transit | Chief of Transit (or Duty Chief) |
| Housing | Chief of Housing (or Duty Chief) |

## NYPD Single Command Incidents

Civil Disturbance, EDP, Explosive Device/Bomb Threat, Hostage, Sniper, Special Event, Suspicious Package, Water/Ice Rescue

## NYPD Core Competencies

Law Enforcement & Investigation, Intelligence Collection & Analysis, Crime Scene Processing/Evidence, Site Management (Perimeter, Traffic, Crowd Control), Site Security & Force Protection, Evacuation, Water Search & Rescue, CBRN/HazMat Site Management/Assessment/Investigation, Collision Investigation, VIP Protection, Arson Investigation (Major Case)
