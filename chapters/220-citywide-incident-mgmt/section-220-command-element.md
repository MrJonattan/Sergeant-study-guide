# Section 220 — Command Element

## P.G. 220-05: CIMS Command Element

**Date Effective:** 08/01/25 | **Last Revision:** R.O. 66 | **Pages:** 15

### Purpose

To outline the duties of the Command Element in CIMS.

### Command Element Responsibilities (Key Steps)

| Step | Responsibility |
|------|---------------|
| 1 | Assume overall incident management responsibility |
| 2 | Obtain briefing; assess situation; notify Desk Officer, Patrol Borough, Operations Unit, Communications |
| 3 | Ensure incident safety; intervene in/suspend unsafe operations |
| 5 | Establish priorities: **Life Safety (#1)**, Site Management, Investigative, Recovery/Restoration |
| 7 | Utilize "Management by Objectives" — determine Incident Objectives, Strategies, Tactical Direction |
| 12 | Select location and establish **Incident Command Post (ICP)** |
| 13 | Manage incident **from the ICP** |
| 14 | Establish appropriate organizational structure |
| 34 | Formulate/approve/authorize one single **Incident Action Plan** (verbal or written) |

### Incident Management Priorities (In Order)

| Priority | Description |
|----------|-------------|
| **#1 — Life Safety** | Immediate priority; supersedes all other operations (absent extraordinary circumstances) |
| #2 — Site Management | Perimeter control, traffic/crowd control, security |
| #3 — Investigative Operations | May run concurrently with Life Safety (with Command Element approval) |
| #4 — Recovery/Restoration | Final mitigation; return to non-emergency; usually after Investigative Operations conclude |

### Incident Objectives Must Be

| Quality | Description |
|---------|-------------|
| **Attainable** | Considering existing limitations and constraints |
| **Measurable** | To determine if being achieved effectively |
| **Adaptable/Flexible** | To provide for alternative strategies |

### Incident Command Post (ICP) Requirements

| Requirement |
|-------------|
| Reasonable distance from incident |
| Safety, health, security, force protection |
| Easy/expeditious access and egress |
| Sufficiently large with room for expansion |
| Permits continuous operations |
| Adequate workspace and utility/communications |
| Generally within the **Site Access Control / Outer Perimeter** |

### ICP Notifications

1. Desk Officer, Precinct of occurrence
2. Patrol Borough Office
3. Communications Section dispatcher/supervisor
4. Operations Unit
5. NYPD Emergency Operations Center (if activated)
6. Other appropriate NYPD units
7. Other appropriate agencies/organizations

### One ICP Rule

- For **every incident** except nuclear weapon detonation, there is **only ONE single ICP**
- Nuclear weapon incidents require **borough-centric ICPs**

### Command Element Structures

| Structure | Description |
|-----------|-------------|
| **Single Command** | One Primary Agency designates one Incident Commander with overall responsibility |
| **Unified Command** | Multiple Primary Agencies each designate an Agency Incident Commander; all co-locate at one ICP; jointly set objectives and create one IAP |

### Span of Control

| Context | Recommended Ratio |
|---------|-------------------|
| General | **1:3 to 1:7** |
| Law enforcement operations | **1:8 to 1:10** |

### Unity of Command

Every subordinate has **only one** designated supervisor.

### Chain of Command

Direction may only be given through the established Chain of Command (absent emergency circumstances).

### Deputy

| Rule | Detail |
|------|--------|
| Qualifications | Same as the principal (knowledge, skills, experience) |
| Must be capable of | Assuming the principal's position |
| May be from | A different agency/jurisdiction |
| Role (flexible) | "Eyes and ears," make notifications, function as relief, assigned specific tasks |

### Transfer of Command

- Should be **face-to-face** if possible
- Incoming Command Element must receive **comprehensive briefing**
- When possible, Transfer of Command should occur at the **start of a new Operational Period**
- Notifications to all Primary Agencies, Supporting Agencies, Staff, Communications

### Demobilization

- Command Element orders demobilization when appropriate
- Ensure all required "After-Action" briefings, reports, meetings, critiques are completed

> **Exam Alert:** **Life Safety is the immediate #1 priority** — it supersedes all other incident management operations (absent extraordinary circumstances).

> **Exam Alert:** CIMS Span of Control: general = **1:3 to 1:7**; law enforcement = **1:8 to 1:10**.

> **Exam Alert:** There is only **ONE Incident Command Post** per incident (except nuclear weapon incidents).

> **Exam Alert:** In Unified Command, Agency Incident Commanders are at **equal authority** regardless of agency rank.

> **Sergeant Focus:** A Sergeant arriving first at a multi-agency incident may serve as the initial Command Element/Incident Commander until relieved through Transfer of Command by a higher-ranking officer.
