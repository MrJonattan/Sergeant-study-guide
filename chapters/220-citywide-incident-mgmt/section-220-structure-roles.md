# Section 220 — CIMS Organizational Structure, Primary Agency Matrix, and Agency Roles

## P.G. 220-01: CIMS Organizational Structure

**Date Effective:** 08/01/13 | **Pages:** 2

### Purpose

To outline the CIMS Organizational Structure.

### Key Concepts

- CIMS structure begins with the **Command Element**
- The structure is **modular** — expanded as needed regardless of incident nature, size, scope, duration, location, or complexity
- Organization may range from Command Element + several Single Resources (small) to Command Element + 3 Command Staff + 5 General Staff Sections (large)

### CIMS Organizational Elements

| Level | Elements |
|-------|----------|
| **Command Element** | Single Command or Unified Command |
| **Command Staff** | Public Information Officer, Safety Officer, Liaison Officer |
| **General Staff — Sections** | Operations, Planning, Logistics, Finance/Administration, Intelligence/Investigations (optional) |

### Operations Section Subordinate Elements

| Element | Activated By |
|---------|-------------|
| Sectors and/or Groups | Operations Section chief |
| Branches | Operations Section chief (if necessary) |

### Other Section Subordinate Elements

| Section | Can Activate |
|---------|-------------|
| Intelligence/Investigations | Groups |
| Logistics | Units, Branches |
| Planning | Units |
| Finance/Administration | Units |

### Organizational Principles

1. Observe recommended **Span of Control** guidelines
2. Activate and staff **only** organizational elements necessary to manage required functions
3. If one individual can manage all functions, **no further organization** is required
4. **Deactivate** elements no longer required — their functions revert to the overhead supervising element

> **Exam Alert:** The CIMS structure is **modular** — only activate what is needed. Non-activated functions are managed by the overhead element.

### CBRN/HazMat Special Rules

| Situation | Command Structure |
|-----------|------------------|
| Actual or suspected crime/terrorism | NYPD **Single Command** |
| No actual or suspected crime/terrorism | **Unified Command** |
| Anticipated/suspected/actual nuclear weapon detonation | **Unified Command** |

> **Exam Alert:** NYPD is the **sole Incident Commander** at CBRN/HazMat incidents with actual or suspected criminal activity/terrorism (Single Command). If NYPD determines **no** crime/terrorism, it switches to Unified Command.

> **Exam Alert:** For auto extrication and boat in distress, the primary agency is determined by **whoever arrives first** (NYPD or FDNY).

## P.G. 220-03: CIMS Agency Core Competencies and Unified Operations Section

**Date Effective:** 08/01/25 | **Last Revision:** R.O. 66 | **Pages:** 3

### Purpose

To define Agency Core Competencies at incidents.

### Key Concept

**Core Competencies** are functional areas of expertise assigned to **one single agency**. Even if multiple agencies can perform similar operations, CIMS assigns the Core Competency to only one agency.

### NYPD Core Competencies

| Core Competency |
|----------------|
| Law Enforcement and Investigation |
| Intelligence Collection and Analysis |
| Crime Scene Processing / Evidence Preservation |
| **Site Management** (Perimeter Control, Traffic Control, Crowd Control) |
| Site Security and Force Protection |
| Evacuation (Area and/or Law Enforcement related) |
| Water Search and Rescue |
| CBRN/HazMat Overall Site Management, Assessment and Investigation |
| Collision Investigation |
| VIP Protection |
| Arson Investigation (Major Case) |

### FDNY Core Competencies

| Core Competency |
|----------------|
| Fire Suppression |
| Pre-hospital Emergency Medical Care |
| Search and Rescue |
| Structural Evacuation |

### Other Agency Core Competencies

| Agency | Core Competency |
|--------|----------------|
| DOB | Damage Assessment (Structural), Building Re-occupancy |
| DDC | Technical Stabilization, Remediation and Demolition |
| DEP | Environmental Monitoring/Mitigation (Chemical) |

## P.G. 220-04: CIMS Agency Roles

**Date Effective:** 08/01/25 | **Last Revision:** R.O. 66 | **Pages:** 2

### Definitions

| Role | Description |
|------|-------------|
| **Primary Agency** | One agency designated to command a Single Command; designates one Incident Commander |
| **Primary Agencies** | Multiple agencies commanding a Unified Command; each designates an Agency Incident Commander |
| **Potential Primary Agencies** | State/federal agencies with jurisdictional authority under specific conditions (airports, military facilities, terrorism) |
| **Supporting Agencies** | Agencies assisting but not commanding the incident |
| **Subject Matter Expert Agencies** | Agencies with specific technical expertise but no personnel in tactical assignments |

### Unified Command Key Rules

- All Agency Incident Commanders **co-locate** at one single ICP
- Jointly set Incident Objectives and create **one single** Incident Action Plan
- All Agency Incident Commanders are at an **equal level of authority** regardless of rank

> **Exam Alert:** In Unified Command, all Agency Incident Commanders are at **equal authority** regardless of their agency rank (Fire Captain = Police Sergeant = EMS Lieutenant in the Command Element).
